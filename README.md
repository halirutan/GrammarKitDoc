# GrammarKit Documentation

Comprehensive user documentation for [GrammarKit](https://github.com/JetBrains/Grammar-Kit), the IntelliJ IDEA plugin for BNF Grammars and JFlex file editing with parser/PSI code generation.

## Overview

This repository contains the source files for GrammarKit's user-facing documentation, built with MkDocs and the Material theme.

## Development

See [DEVELOPMENT.md](DEVELOPMENT.md) for setup and build instructions.

## Quick Start

```bash
# Install dependencies
uv sync

# Run development server
uv run mkdocs serve

# Build documentation
uv run mkdocs build
```
