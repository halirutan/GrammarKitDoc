## 3. Deprecated Attributes

**Source: `attributeDescriptions/generateFirstCheck.html`**

`generateFirstCheck` is marked as deprecated:
> Deprecated: use **generate** attribute

The `generate=[first-check=N]` syntax supersedes the standalone attribute. Similarly, `generatePsi`, `generateTokens`, `generateTokenAccessors`, and `extendedPin` are all superseded by the consolidated `generate` attribute.

## 4. Java Version Default Change

**Source: `CHANGELOG.md:67`, `GenOptions.java:131`**

- Pre-2020.3: default Java version was 6 (then 8)
- 2020.3+: default switched to Java 11
- Effect: lambdas generated by default, method references used where applicable

## 7. KnownAttribute.java as Attribute Catalog

**Source: `src/org/intellij/grammar/KnownAttribute.java:29-72`**

All known attributes are defined here. When new attributes are added or existing ones change defaults, this file is the single source of truth. New attributes in recent versions:
- `extraRoot` (2019.1)
- `hooks` (1.4.0)
- `fallbackStubElementType` (added for stub support)

## 8. API Changes

**Source: `CHANGELOG.md`**

Notable API changes requiring migration:
- 2017.1: Merged AST and PSI trees — `CompositePsiElement` inheritors supported
- 2020.3: `@Override` on visitor methods, type annotations (ASM-based)
- 2021.1: Java-related extensions moved to optional config
- 2022.3.1: Constructor injection removed (IJ 2023.1 compatibility)
